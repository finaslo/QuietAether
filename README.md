# QuietAether
Панель управления машиниста предназначенная для Raspberry Pi. Является частью проекта "Тихий Эфир" РУТ МИИТ.

# Установка
1. Создать venv пространство с именем QAvenv
```python3 -m venv QAvenv```
2. Установить требования в пространство
```source QAvenv/bin/activate```
```pip install requirements.txt```

# Использование
Запустите ```StartQuietAether.sh```
